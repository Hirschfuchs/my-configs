Zunächst werden alle IDE- und Entwicklungstools gemeinsam im Modul `entwicklung` zusammengetragen.
Später soll es eine Unterscheidung geben nach:

* Wichtige IDEs (Jetbrains Toolbox)
* Nebenläufige IDEs (bspw. Arduino)
* Entwicklungstools (bspw. Putty)
