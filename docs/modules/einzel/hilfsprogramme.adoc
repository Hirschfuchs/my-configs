Das erste Modul, das in diesem Projekt erstellt wird,
ist das Modul `hilfsprogramme`.
Dies sind Standardfunktionalitäten, die ich auf einem System
zur reibungslosen Verwendung vorfinden möchte.
Gleichzeitig sind es nicht so tiefenessenzielle Dinge,
wie das Betriebssystem oder der Bootmanager,
weshalb hier erstmal gut die Migration von _Pacman_ auf _Nix_ getestet werden kann.
